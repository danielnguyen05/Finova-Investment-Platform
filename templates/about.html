About FINOVA

FINOVA is a personal finance platform created by Alice and Bob to showcase their expertise in finance and economics.

The platform offers several sophisticated features, including:

  • - Integration of the Alpha Vantage API for real-time and accurate stock market data.
  • - A built-in, fine-tuned chatbot powered by Ollama 3.2 to answer finance and economics-related questions.
  • - An interactive investment growth plot that estimates portfolio value using finance principles and linear programming (CBC model).

FINOVA's frontend is built using HTML, CSS, and JavaScript, while the backend leverages Python (Flask, pandas).
